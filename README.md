# TripLog Frontend

TripLog is an all-in-one travel companion platform that brings together budget management, route planning, group collaboration, and AI-driven insights — all in a single service.

This repository contains the frontend application for TripLog, built with React and Vite.

## Tech Stack

- **Framework**: React (Vite)
- **Language**: TypeScript
- **Map (3D)**: CesiumJS — 3D globe entry visualization
- **Map (2D)**: Google Maps API — route mapping and visualization
- **Styling**: CSS

## Setup

```bash
npm install
npm run dev
```
